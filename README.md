# Very Bad Cat Clicker Game

[![Godot](https://img.shields.io/badge/Engine-Godot-blue)](#)
[![Platform](https://img.shields.io/badge/Platform-Web-red)](#)

A casual and simple *idle clicker* game. In this game, your main goal is simple: click a funny cat to gather as many coins as possible, then spend those coins to buy upgrades!

techstack:
	gdscript
